GHOST_CHAIN_WHITE_PAPER

ABSTRACT: An ephemeral, decentralized communication protocol designed for zero-trace data exchange. Ghost Chain exists solely in volatile memory (RAM).

SIGNALING: Utilizes deterministic SHA-256 peer-id derivation to eliminate the need for centralized discovery relays.

SECURITY: Mandatory end-to-end encryption via AES-256-GCM. Payloads are encrypted locally before transmission. The Secret Key is never broadcast.

PERSISTENCE: Non-existent. Closing the browser tab acts as a hardware-level kill switch, permanently erasing all ledger history.
